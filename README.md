# 📊 Data Analysis with Python & SQL

Projeto de estudos focado em análise de dados utilizando Python e SQL.

---

## 🚀 Tecnologias e Ferramentas
- Python
- SQL (SQLite)
- VS Code
- Deepnote

---

## 📚 O que estou praticando
- Manipulação de dados com Python
- Estruturas de listas e matrizes
- Consultas SQL (SELECT, JOIN, GROUP BY, HAVING)
- Modelagem de banco de dados
- Integração entre Python e SQL

---

## 🎯 Objetivo
Desenvolver habilidades práticas em análise de dados, resolvendo exercícios baseados em cenários reais como filmes, avaliações e bilheteria.

---

## 📂 Estrutura do projeto
- `main.py` → exercícios em Python + SQL
- `cinema.sqlite` → banco de dados utilizado

---

## 📌 Observações
Projeto desenvolvido utilizando VS Code e Deepnote como ambiente de desenvolvimento.
